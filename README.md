# Plunder

可配置、插件化的爬虫引擎，二次开发框架；大数据累积项目。

支持单例、分布式部署。

Configurable, pluginable crawler engine, secondary development framework.

## Version

0.8 beta.

## Modules：

Engine、Scheduler、ExcludeDuplicateFilter、Downloader、Analyzer、Producer、ResultPipeline、Log、Configuration
