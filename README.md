# Plunder

可配置、插件化的爬虫引擎，二次开发框架。

Configurable, pluginable crawler engine, secondary development framework.

## Modules：

Engine、Scheduler、ExcludeDuplicateFilter、Downloader、Analyzer、Producer、ResultPipeline、Log、Storage
